### Document Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!-- Content goes here -->
```

### Forms (VERY IMPORTANT!)

#### Complete Form Example

```html
<form id="myForm" onsubmit="handleSubmit(event)">
    
    <!-- Text Input -->
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" placeholder="Enter name" required>
    
    <!-- Email Input -->
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <!-- Password Input -->
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required>
    
    <!-- Number Input -->
    <label for="age">Age:</label>
    <input type="number" id="age" name="age" min="1" max="120">
    
    <!-- Tel/Phone Input -->
    <label for="phone">Phone:</label>
    <input type="tel" id="phone" name="phone" pattern="[0-9]{10}">
    
    <!-- Radio Buttons (select one) -->
    <p>Gender:</p>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>
    
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label>
    
    <!-- Checkboxes (select multiple) -->
    <p>Interests:</p>
    <input type="checkbox" id="coding" name="interests" value="coding">
    <label for="coding">Coding</label>
    
    <input type="checkbox" id="music" name="interests" value="music">
    <label for="music">Music</label>
    
    <!-- Select Dropdown -->
    <label for="country">Country:</label>
    <select id="country" name="country">
        <option value="">Select country</option>
        <option value="india">India</option>
        <option value="usa">USA</option>
        <option value="uk">UK</option>
    </select>
    
    <!-- Textarea -->
    <label for="message">Message:</label>
    <textarea id="message" name="message" rows="5" cols="30"></textarea>
    
    <!-- Date Input -->
    <label for="dob">Date of Birth:</label>
    <input type="date" id="dob" name="dob">
    
    <!-- Submit Button -->
    <button type="submit">Submit</button>
    
    <!-- Regular Button -->
    <button type="button" onclick="clearForm()">Clear</button>
    
    <!-- Reset Button -->
    <button type="reset">Reset</button>
</form>
```

#### Input Attributes

- `required` - Field must be filled
- `placeholder` - Hint text
- `value` - Default value
- `disabled` - Cannot be edited
- `readonly` - Can see but not edit
- `min`, `max` - For numbers
- `pattern` - Validation pattern (regex)

## üé® CSS BASICS

### How to Add CSS

#### 1. Inline CSS (in HTML element)

```html
<p style="color: red; font-size: 20px;">Red text</p>
```

#### 2. Internal CSS (in `<head>`)

```html
<head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>
```

#### 3. External CSS (separate file)

```html
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

### CSS Selectors

```css
/* Element selector */
p {
    color: blue;
}

/* Class selector (use . before class name) */
.myClass {
    color: red;
}

/* ID selector (use # before id) */
#myId {
    color: green;
}

/* Multiple selectors */
h1, h2, h3 {
    font-family: Arial;
}

/* Descendant selector (space) */
div p {
    /* All p inside div */
    color: purple;
}

/* Child selector (>) */
div > p {
    /* Direct children only */
    color: orange;
}

/* Pseudo-classes */
a:hover {
    /* When mouse over link */
    color: red;
}

input:focus {
    /* When input is focused */
    border: 2px solid blue;
}

button:active {
    /* When button is clicked */
    background-color: green;
}
```

### CSS Properties

#### Colors

```css
.element {
    color: red;                    /* Text color */
    background-color: #ff0000;     /* Background (hex) */
    background-color: rgb(255, 0, 0);  /* RGB */
    background-color: rgba(255, 0, 0, 0.5);  /* RGBA (with transparency) */
}
```

#### Text Styling

```css
.text {
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;              /* or normal, 100-900 */
    font-style: italic;             /* or normal */
    text-align: center;             /* left, right, center, justify */
    text-decoration: underline;     /* underline, line-through, none */
    text-transform: uppercase;      /* uppercase, lowercase, capitalize */
    letter-spacing: 2px;
    line-height: 1.5;
}
```

#### Box Model

```css
.box {
    /* Content size */
    width: 300px;
    height: 200px;
    
    /* Padding (inside border) */
    padding: 20px;                  /* All sides */
    padding: 10px 20px;             /* top/bottom, left/right */
    padding: 10px 20px 15px 25px;  /* top, right, bottom, left */
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 15px;
    padding-left: 25px;
    
    /* Border */
    border: 2px solid black;
    border-radius: 10px;            /* Rounded corners */
    
    /* Margin (outside border) */
    margin: 20px;                   /* Same as padding */
    margin: 0 auto;                 /* Center horizontally */
}
```

#### Display & Visibility

```css
.element {
    display: block;      /* Takes full width, starts on new line */
    display: inline;     /* Takes only needed width, doesn't break line */
    display: inline-block;  /* Inline but can have width/height */
    display: none;       /* Completely hidden, no space taken */
    
    visibility: hidden;  /* Hidden but space is preserved */
    visibility: visible;
}
```

#### Positioning

```css
.element {
    position: static;    /* Default, normal flow */
    position: relative;  /* Relative to normal position */
    position: absolute;  /* Relative to parent */
    position: fixed;     /* Fixed to viewport */
    position: sticky;    /* Sticky on scroll */
    
    /* Use with position */
    top: 10px;
    right: 20px;
    bottom: 15px;
    left: 25px;
    z-index: 10;        /* Stack order */
}
```

#### Flexbox (Modern Layout)

```css
.container {
    display: flex;
    
    /* Direction */
    flex-direction: row;            /* Default: left to right */
    flex-direction: column;         /* Top to bottom */
    
    /* Alignment */
    justify-content: center;        /* Main axis (horizontal if row) */
    justify-content: space-between;
    justify-content: flex-start;
    justify-content: flex-end;
    
    align-items: center;            /* Cross axis (vertical if row) */
    align-items: flex-start;
    align-items: flex-end;
    
    /* Wrap */
    flex-wrap: wrap;
    
    /* Gap between items */
    gap: 10px;
}

.flex-item {
    flex: 1;                        /* Grow to fill space */
    flex-shrink: 0;                 /* Don't shrink */
}
```

#### Backgrounds

```css
.element {
    background-color: #f0f0f0;
    background-image: url('image.jpg');
    background-size: cover;         /* Fit entire area */
    background-position: center;
    background-repeat: no-repeat;
}
```

#### Shadows & Effects

```css
.element {
    box-shadow: 2px 2px 10px rgba(0,0,0,0.3);
    text-shadow: 1px 1px 2px black;
    opacity: 0.8;                   /* 0 to 1 */
    transform: rotate(45deg);
    transform: scale(1.5);
    transition: all 0.3s ease;      /* Smooth changes */
}
```

## ‚ö° JAVASCRIPT BASICS

### Variables

```javascript
// var (old way, avoid)
var x = 5;

// let (can change value)
let name = "John";
name = "Jane";  // OK

// const (cannot change value)
const PI = 3.14;
// PI = 3.15;  // ERROR!
```

### Data Types

```javascript
// String
let text = "Hello";
let text2 = 'World';
let text3 = `Hello ${name}`;  // Template literal

// Number
let age = 25;
let price = 99.99;

// Boolean
let isActive = true;
let isLoggedIn = false;

// Array
let fruits = ["apple", "banana", "orange"];
let numbers = [1, 2, 3, 4, 5];

// Object
let person = {
    name: "John",
    age: 30,
    city: "Mumbai"
};

// Null and Undefined
let empty = null;
let notDefined;  // undefined
```

### Operators

```javascript
// Arithmetic
let sum = 5 + 3;        // 8
let diff = 10 - 4;      // 6
let product = 4 * 3;    // 12
let quotient = 15 / 3;  // 5
let remainder = 10 % 3; // 1
let power = 2 ** 3;     // 8

// Increment/Decrement
let count = 0;
count++;  // 1
count--;  // 0

// Comparison
5 == "5"    // true (loose equality)
5 === "5"   // false (strict equality)
5 != "5"    // false
5 !== "5"   // true
5 > 3       // true
5 < 3       // false
5 >= 5      // true

// Logical
true && false   // AND: false
true || false   // OR: true
!true          // NOT: false

// String concatenation
let fullName = "John" + " " + "Doe";
```

### Conditionals

```javascript
// if-else
if (age >= 18) {
    console.log("Adult");
} else if (age >= 13) {
    console.log("Teenager");
} else {
    console.log("Child");
}

// Ternary operator
let status = age >= 18 ? "Adult" : "Minor";

// Switch
switch(day) {
    case "Monday":
        console.log("Start of week");
        break;
    case "Friday":
        console.log("End of week");
        break;
    default:
        console.log("Midweek");
}
```

### Loops

```javascript
// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}

// While loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// For...of (arrays)
let colors = ["red", "green", "blue"];
for (let color of colors) {
    console.log(color);
}

// forEach (arrays)
fruits.forEach(function(fruit) {
    console.log(fruit);
});

// Or with arrow function
fruits.forEach(fruit => console.log(fruit));
```

### Functions

```javascript
// Function declaration
function greet(name) {
    return "Hello " + name;
}

// Function expression
const add = function(a, b) {
    return a + b;
};

// Arrow function (modern)
const multiply = (a, b) => {
    return a * b;
};

// Arrow function (short form)
const square = x => x * x;

// Calling functions
let result = add(5, 3);      // 8
let greeting = greet("John"); // "Hello John"
```

### Arrays

```javascript
let fruits = ["apple", "banana", "orange"];

// Access elements
fruits[0]  // "apple"
fruits[1]  // "banana"

// Array length
fruits.length  // 3

// Add elements
fruits.push("grape");        // Add to end
fruits.unshift("mango");     // Add to start

// Remove elements
fruits.pop();                // Remove from end
fruits.shift();              // Remove from start

// Find element
fruits.indexOf("banana");    // 1
fruits.includes("apple");    // true

// Loop through
fruits.forEach(fruit => {
    console.log(fruit);
});

// Map (transform array)
let lengths = fruits.map(fruit => fruit.length);

// Filter (get matching items)
let longFruits = fruits.filter(fruit => fruit.length > 5);

// Join array to string
fruits.join(", ");  // "apple, banana, orange"
```

### Objects

### DOM Manipulation (MOST IMPORTANT!)

#### Get Elements

```javascript
// By ID (most common and fastest)
let element = document.getElementById("myId");

// By class name
let elements = document.getElementsByClassName("myClass");

// By tag name
let paragraphs = document.getElementsByTagName("p");

// Query selector (CSS selector)

let elements = document.querySelectorAll(".myClass"); // All matches
```

#### Modify Content

```javascript
// Text content
element.textContent = "New text";
element.innerText = "New text";

// HTML content
element.innerHTML = "<b>Bold text</b>";

// Form values
inputElement.value = "New value";
let userInput = inputElement.value;  // Get input value
```

#### Modify Styles

```javascript
element.style.color = "red";
element.style.backgroundColor = "blue";

```

#### Modify Attributes

```javascript

// Class manipulation
```

#### Event Handlers

```javascript
// Method 1: Inline HTML
<button onclick="myFunction()">Click</button>

// Method 2: Property
button.onclick = function() {
    alert("Clicked!");
};

// Method 3: addEventListener (best)
button.addEventListener("click", function() {
    alert("Clicked!");
});

// Arrow function
button.addEventListener("click", () => {
    alert("Clicked!");
});

// Common events
element.addEventListener("click", handler);
element.addEventListener("dblclick", handler);
input.addEventListener("change", handler);
input.addEventListener("input", handler);    // Every keystroke
input.addEventListener("focus", handler);
input.addEventListener("blur", handler);
form.addEventListener("submit", handler);
element.addEventListener("mouseover", handler);
element.addEventListener("mouseout", handler);
```

#### Form Handling

```javascript
const form = document.getElementById("myForm");

form.addEventListener("submit", function(event) {
    event.preventDefault();  // IMPORTANT: Stop page reload
    
    // Get form values
    const name = document.getElementById("name").value;
    const email = document.getElementById("email").value;
    
    // Validation
    if (name === "") {
        alert("Name is required");
        return;
    }
    
    // Do something with data
    console.log({name, email});
    
    // Clear form
    form.reset();
});
```

### Common Patterns

#### Show/Hide Element

```javascript
function toggleElement(id) {
    const element = document.getElementById(id);
    if (element.style.display === "none") {
        element.style.display = "block";
    } else {
        element.style.display = "none";
    }
}
```

#### Calculate Total

```javascript
function calculateTotal() {
    const checkboxes = document.querySelectorAll('input[type="checkbox"]:checked');
    let total = 0;
    
    checkboxes.forEach(checkbox => {
        total += parseFloat(checkbox.value);
    });
    
    document.getElementById("total").textContent = total.toFixed(2);
}
```

#### Validation

```javascript
function validateEmail(email) {
    const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return regex.test(email);
}

function validatePhone(phone) {
    const regex = /^\d{10}$/;
    return regex.test(phone);
}

// Usage
if (!validateEmail(email)) {
    alert("Invalid email");
    return false;
}
```

#### Clear Form

```javascript
function clearForm() {
    document.getElementById("myForm").reset();
}
```

#### Dynamic Content

```javascript
// Add element
function addItem(text) {
    const li = document.createElement("li");
    li.textContent = text;
    document.getElementById("list").appendChild(li);
}

// Remove element
function removeItem(element) {
    element.remove();
}
```

## üéØ Problem-Solving Approach

### Any Question Can Be Broken Down Into

1. **HTML Structure** - What elements do I need?
2. **CSS Styling** - How should it look?
3. **JavaScript Logic** - What should happen?

### Step-by-Step Process

```
1. READ the question carefully
2. IDENTIFY what's needed:
   - New HTML elements?
   - CSS changes?
   - JavaScript functions?
3. PLAN your approach
4. WRITE the code
5. TEST it works
6. DEBUG if needed
```

### Debugging Tips

```javascript
// Print to console
console.log("Value:", myVariable);

// Check if element exists
if (element === null) {
    console.log("Element not found!");
}

// Check in browser:
// - Open Console (F12)
// - Look for red errors
// - Click on error to see line number
```

## ‚úÖ Quick Reference Checklist

### For ANY Question, Ask

- [ ] Do I need new HTML elements?
- [ ] Do I need to modify CSS?
- [ ] Do I need JavaScript to make it interactive?
- [ ] Do I need to handle form submission?
- [ ] Do I need validation?
- [ ] Do I need calculations?
- [ ] Do I need to show/hide things?

### Common Tasks

- **Add element**: Write HTML
- **Style element**: Write CSS (inline, internal, or external)
- **Make interactive**: Write JavaScript function
- **Get user input**: `document.getElementById("id").value`
- **Show result**: `element.textContent = result`
- **Validate**: Use regex or if conditions
- **Calculate**: Use math operators and loops
- **Toggle**: Check current state and switch
